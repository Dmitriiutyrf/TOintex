# ИНТЕКС-СБ — Система управления техническим обслуживанием

Современное веб-приложение для специалистов по техническому обслуживанию пожарной сигнализации и систем безопасности. Позволяет вести учет объектов, отслеживать сроки ТО и управлять историей обслуживания.

## 🚀 Основные возможности

- **Авторизация**: Вход через Telegram-виджет или по номеру телефона (с поддержкой демо-режима).
- **Управление объектами**: Удобная группировка объектов по заказчикам (аккордеон).
- **Контроль сроков ТО**: Автоматическое отслеживание просрочек (более 30 дней) с наглядной цветовой индикацией (зеленые/красные плашки).
- **Уведомления**: Глобальный баннер и счетчики просроченных объектов для каждого заказчика.
- **История обслуживания**: Добавление новых записей о ТО с датой и комментариями.
- **Адаптивный дизайн**: Удобный интерфейс для мобильных устройств и ПК (Tailwind CSS + Framer Motion).

## 🛠 Технологический стек

- **Frontend**: React 19, TypeScript, Vite
- **Стилизация**: Tailwind CSS (v4), clsx, tailwind-merge
- **Анимации**: Framer Motion
- **Иконки**: Lucide React
- **Уведомления**: Sonner

## 📦 Установка и локальный запуск

1. Клонируйте репозиторий:
   ```bash
   git clone <url-вашего-репозитория>
   cd tointex
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите сервер для разработки:
   ```bash
   npm run dev
   ```

4. Откройте приложение в браузере по адресу `http://localhost:3000`.

## 🤖 Настройка Telegram-авторизации

Для работы реальной авторизации через Telegram:
1. Создайте бота через [@BotFather](https://t.me/BotFather) в Telegram.
2. Привяжите домен вашего развернутого приложения к боту с помощью команды `/setdomain`.
3. Введите `username` вашего бота на экране входа в приложении.

## 📄 Структура проекта

- `/src/components` — переиспользуемые UI-компоненты (например, виджет Telegram).
- `/src/App.tsx` — главный компонент приложения, логика авторизации и отображения списков.
- `/src/constants.ts` — тестовые данные (MOCK_OBJECTS) и начальное состояние базы.
- `/src/types.ts` — TypeScript-интерфейсы.
- `/src/index.css` — глобальные стили и переменные Tailwind.
